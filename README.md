## Dependencies

This software requires a mix of **Python**, **R**, and **Linux command-line tools**. 

A comprehensive list of all required software and packages is provided in [requirements.txt](requirements.txt). Please ensure the following dependencies are installed and added to your system PATH:

* **Linux Tools:** See `requirements.txt` (e.g., samtools, bedtools).
* **Python (3.8+):** Core modules listed in `requirements.txt`.
* **R (4.0+):** Required CRAN/Bioconductor libraries listed in `requirements.txt`.

Please verify that all external tools are executable from your command line before running the pipeline.
